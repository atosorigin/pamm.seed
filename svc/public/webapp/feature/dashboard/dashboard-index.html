Start Page - Introduction

Click on the sample tab to see a sample list implementation. On the "Manage Project" page - you will a list of sample data. This list of data comes from the repository and the (mock) DAL service. See service/repository and service/dal.

The DAL service is where you implement connection to the back end. Currently, there is a mock implementation for demonstration purpose. (Note: it is not good practice to put mocks alongside real code. Future releases of this seed will use external configuration to switch.

Both the DAL and Repository services use decorator to implement separation of concerns. See examples in the service directory.

Technology Stack

  • AngularJS 1.4
  • Angular UI Router (replaces default router)
  • Bootstrap 3
  • Bootstrap Dialog
  • lodash - Javascript utilities libary

Test Stack

  • Jasmine 2.0 (unit)
  • karma (auto)
  • Protractor (e2e)
Note that Karma and protractor require node.js
